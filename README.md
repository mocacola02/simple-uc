# Simple UnrealScript (simple-uc)
A simple yet powerful VS Code/Codium extension adding syntax highlighting and Intellisense for UE'1 UnrealScript.

*Notice: This extension is currently in early development. Only syntax highlighting is (partially) functional right now.*

## Install
The following node modules are required:
- @types/node
- @types/vscode
- typescript
- undici-types
